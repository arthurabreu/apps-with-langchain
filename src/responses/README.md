# Responses & Exports Directory

This directory stores all outputs from the LangChain application.

## Structure

```
responses/
├── excel_exports/      # Excel files generated by "LLM to Excel Agent"
│   ├── llm_output_*.{json,md}    # Intermediate LLM outputs
│   └── llm_report_*.xlsx         # Final Excel files
└── README.md          # This file
```

## File Naming Convention

### Excel Exports
- **LLM Output Files**: `llm_output_YYYYMMDD_HHMMSS.{json,md}`
- **Excel Reports**: `llm_report_YYYYMMDD_HHMMSS.xlsx`

Each file is timestamped for easy tracking and organization.

## How Files Are Saved

1. When you run the **"LLM to Excel Agent"**:
   - The LLM generates a response (JSON or Markdown)
   - Response is saved to `excel_exports/llm_output_*.{json,md}`
   - Data is converted to an Excel spreadsheet
   - Final report is saved to `excel_exports/llm_report_*.xlsx`

2. Timestamps use Brazil/São Paulo timezone for consistency with the application logs

## Tips

- Excel files are ready to use immediately after generation
- Keep this directory for reference and archival purposes
- You can safely delete old exports if you need to free up space
- File paths are displayed in the console when generated

## Location

Full path: `<project_root>/responses/excel_exports/`
